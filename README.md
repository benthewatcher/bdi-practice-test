# DGI Practice Test — Certificate in Board Directorship

Interactive self-scoring practice test (GCC BDI exam style) covering
Module 1 (Corporate Governance Essentials), Module 2 (The Effective Director)
and Module 4 (Finance, Accounting & Corporate Reporting). Module 3 is reserved
and will unlock when its questions are added.

Static site — `index.html` is the entire app (no build step).

## Deploy on Vercel
1. Push this folder to a GitHub repo.
2. In Vercel: New Project → import the repo.
3. Framework preset: **Other**. Build command: none. Output dir: `.` (root).
4. Deploy. Done.
